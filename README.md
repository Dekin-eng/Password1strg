## 📖 About

**PASS1STRG** is a professional password strength auditing tool that analyzes passwords, detects weaknesses, checks for password reuse, and generates comprehensive security reports. Available with both a modern GUI and a command-line interface.

Perfect for security audits, password policy enforcement, and personal password management.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔍 **Password Strength Analysis** | Evaluates passwords based on length, complexity, and patterns |
| 📊 **Real-time Strength Meter** | Live feedback as you type |
| 🔁 **Password Reuse Detection** | Identifies duplicate passwords across accounts |
| 📋 **Copy to Clipboard** | Easy sharing of analysis results |
| 📄 **Report Generation** | HTML and TXT reports with detailed analysis |
| 🎨 **Modern Dark Theme** | Professional, easy-on-the-eyes interface |
| 💻 **Dual Interface** | GUI for daily use + CLI for automation |
| 📁 **Sample Data** | Built-in test data for demonstrations |
